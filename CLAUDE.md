# Content

This is a generic directory for hosting self-contained research and strategy documentation as HTML files.

## Directory structure

- `index.html` — Navigation index; lists all content files with links
- `*.html` — Individual research/strategy documents

## Conventions

- Each document is a self-contained, styled HTML file (no external build step or framework)
- Add every new HTML file to `index.html` manually

## When adding a new document

1. Create the `.html` file in this directory
2. Add a corresponding `<li><a>` entry in `index.html`
